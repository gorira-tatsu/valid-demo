# 掲示板 RDD

## 概要
- 本ディレクトリは、簡単な掲示板アプリの要件定義と仕様をまとめるためのもの。
- 想定対象は MVP（Minimum Viable Product）であり、まずは基本的な投稿と閲覧を成立させる。
- 実装方式や使用技術は未確定のため、本資料では機能要件・業務ルール・画面/API 観点の仕様を整理する。

## 想定する MVP 機能
- 投稿の一覧表示
- 投稿の新規作成
- 投稿の詳細表示
- 投稿の編集、削除
- 投稿へのコメント追加、一覧表示

## 本資料の前提
- ユーザー認証は MVP では必須としない。
- 投稿者名は入力式とし、匿名投稿を許可する。
- 管理者機能、通報機能、画像投稿、検索、高度な権限制御はスコープ外とする。
